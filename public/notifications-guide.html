Enable Notifications

Get notified instantly when a broadcast goes live. Choose your device below for setup instructions.

iPhone / iPad (iOS 16.4+)

Apple requires you to add UmmahCast to your Home Screen first. Push notifications only work when opened from the Home Screen icon — not from Safari directly.

1
Open Safari — it must be Safari, not Chrome or another browser. Go to ummahcast.com
2
Tap the Share button — the square with an upward arrow (↑) at the bottom of the screen
3
Tap "Add to Home Screen" — scroll down in the share sheet if you don't see it. Tap "Add" to confirm.
4
Open from Home Screen — find the UmmahCast icon on your Home Screen and tap it. It will open as a standalone app.
5
Tap "🔔 Push Notifications" — then tap "Allow" on the iOS permission prompt. Done!

Troubleshooting

  • Must use Safari — other iOS browsers don't support Add to Home Screen with push
  • Must open from the Home Screen icon — push doesn't work in regular Safari
  • Requires iOS 16.4 or later — check Settings → General → About
  • Check Settings → Notifications → UmmahCast to verify notifications are enabled

Android (Chrome)

Android Chrome supports push notifications directly — no special setup needed.

1
Open Chrome — go to ummahcast.com
2
Tap "🔔 Push Notifications" — the button on the main page
3
Tap "Allow" — on the Chrome permission prompt. That's it!

If notifications were previously blocked

  • Tap the lock icon next to the URL → Permissions → Notifications → Allow
  • Or go to Chrome Settings → Site Settings → Notifications → find UmmahCast → Allow

Optional: Tap the three-dot menu → "Install app" to add UmmahCast to your Home Screen for the best experience.

Desktop (Chrome / Edge / Firefox)

1
Visit ummahcast.com — in any modern browser
2
Click "🔔 Push Notifications"
3
Click "Allow" — on the browser permission prompt

Note

  • Brave browser blocks push notifications by default — go to brave://settings/privacy and enable "Use Google services for push messaging"
  • macOS users may need to allow notifications in System Settings → Notifications → Chrome/Firefox
